Apache APISIX 3.9.0 正式发布

我们很高兴地宣布 Apache APISIX 3.9.0 版本已经发布，带来了一系列新功能、修复、以及相关用户体验优化。

•
2024年3月29日
Community

社区月报 (01.29 - 02.29)

最近，我们新增并改进了 Apache APISIX 的部分功能，如新增 http-dubbo 插件、limit-req 和 limit-conn 插件新增支持 redis 和 redis-cluster、日志类插件增加支持 include_req_body 和 include_resp_body等。有关更多功能新亮点，请阅读本期月报。

最近，Apache APISIX 新增了 ocsp-stapling 插件，详情请阅读本期双周报。

2024年1月31日
Community

社区双周报 (01.01 - 01.14)

最近，我们新增并改进了 Apache APISIX 的部分功能，如日志相关插件新增 includereq_body 相关选项、支持从源代码一键编译安装 apisix 和 apisix-runtime、response-rewrite 插件在使用 filters.regex 选项时支持 Brotli 压缩，以及当使用 radixtree_uri_with_parameter 路由引擎时支持 uri_param 变量。详情请阅读本期双周报。

Apache APISIX 3.8.0 正式发布

我们很高兴地宣布 Apache APISIX 3.8.0 版本已经发布，带来了一系列新功能、修复、以及相关用户体验优化。

•
2024年1月15日
Community

社区双周报 (12.18 - 12.31)

最近，我们新增并改进了 Apache APISIX 的部分功能，如 limit-count 插件配置支持环境变量、response-rewrite 插件在使用 filters.regex 选项时支持 gzip、升级 OpenSSL 1.1.1 至 OpenSSL 3.0 版本。详情请阅读本期双周报。

2024年1月3日
Community

社区双周报 (12.04 - 12.17)

最近，我们新增并改进了 Apache APISIX 的部分功能，如新增 multi-auth 和 brotli 插件、openid-connect 插件新增更多属性、CORS 插件支持 Timing-Allow-Origin 响应头、使用新的 lua-resty-events 作为默认事件机制。详情请阅读本期双周报。
